How To Use Images

Images can be used with the turtle module. 

import turtle
screen = turtle.Screen()
# set the screen background
screen.bgpic("filename.png")

# Or, set the shape of a turtle
screen.addshape("filename.png")
tina = turtle.Turtle()
tina.shape("filename.png")
